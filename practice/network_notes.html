Network Notes

Vocabularly

Protocol
A set of rules that governs how traffic flows in a network.
Packet
Small segments of data that larger chunks of data are split up into.
Header
Meta data attached to a packet that contains information such as the packets origin, destination, and which # of packets.
Payload
The actual data contained in a packet, the content of the message.
Router
Special kind of computer that routes packets.
Routing Table
Data table that contains routes to network destinations.

Problems That Occur on a Packet Switched Network

  • Lost Packets
    • Packets can get lost for a variety of reasons.
  • Overloaded Computers
    • Sometimes, a computer can get overloaded with packets, at which point it usually shuts down and loses all of the packets it had at the time.
  • Changing Network Structure
    • The structure of the internet is constantly changing for a variety of reasons, from power outages to new computers coming online.

Protocols We Came Up With in Class

Packet Headers
Adding a header to packets specifying which # packet it was, which address sent it, and where it was going to.
This helps by telling the recipient how to put the packets together, and it helps the computers on the network that it goes through get the packets where they need to go.
Who are you?
Asking neighbors for IP address, helps map the network.
Request Network Information
Asking your neighbors for who their neighbors are, helps the sender send the packet in the right direction.
