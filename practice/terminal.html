ls is a command that shows you the files in your current directory. The most useful forms are displayed below.

ls

This command will display all non-hidden files and subdirectories within your current directory. It is used because it is faster to type than the other form that is often used.

ls -a

This command will display all file and subdirectories, including all hidden files and subdirectories, that are within your current directory.

The output of a ls command would look like what is displayed below.

index.html minimal.html

The output of the ls -a command would look the same, but with hidden files included among the displayed files.
